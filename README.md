# Chat Phone

Shinsekai 手机模拟器插件。在聊天界面中嵌入一个完整的手机组件，支持短信、通话、浏览器、音乐控制等功能。

## 功能

### 短信
- 与通讯录中的角色互发短信
- 支持主动联系（角色定时发消息/打电话，频率可配）
- 未读角标 + 震动提醒

### 通话
- 语音通话 + 视频通话（立绘展示）
- 通话记录（时长、方向）
- 拨号键盘

### 应用
| App | 功能 |
|---|---|
| 📱 通话 | 最近通话 / 联系人 / 拨号 |
| 💬 短信 | 聊天列表 / 聊天气泡 |
| 🌐 浏览器 | LLM 搜索 / 网页抓取 |
| 🎵 音乐 | 本地播放器控制 / 一起听 |
| 🎤 录音 | 捕获 TTS 音频并合并 |
| ⚙️ 设置 | 勿扰 / 主题 / 频率配置 |

### 黑客模式
在剧情中对角色手机安装监控程序后，可实时查看该角色的短信、通话、浏览器搜索等私密活动。

### 病娇彩蛋
- 自动检测角色设定中的病娇关键词
- 角色在对话中建立手机被动手脚后，通话挂断会被拦截
- 情绪崩溃时触发，挂断按钮点击 8 次才能挂掉
- 注入角色已知的用户私密情报（浏览器历史、短信记录、通话记录）

## 主题

5 种马卡龙配色可选：白 / 婴儿蓝 / 浅粉 / 淡紫 / 黑

## 数据存储

`data/plugins/com.shinsekai.chat_phone/`

按会话隔离，包含通讯录、短信、通话记录、浏览器历史等。

## LLM 工具

| 工具 | 说明 |
|---|---|
| `exchange_contacts` | 交换联系方式，角色加入通讯录 |
| `send_sms` | 以角色身份发送短信 |
| `bug_character_phone` | 在角色手机上安装监控程序 |
